# Student Dashboard

A **fully responsive single-page Student Dashboard** built using **Bootstrap 5**.  
Demonstrates professional UI with cards, tables, progress bars, alerts, and sidebar navigation.

## Features
- Sidebar with active link highlight and mobile toggle
- Dashboard cards: Total Subjects, Attendance, GPA
- Attendance progress bar
- Marks table
- Alert for notices
- Bootstrap icons for UI polish
- Fully responsive layout

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons
